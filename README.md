# Retífica de Motores Irecê

Site institucional da **Retífica de Motores Irecê** (Jacodiesel) — retífica de motores
linha leve (álcool/gasolina) e pesada (diesel) em Irecê, Bahia.

Site estático (HTML/CSS/JS), sem dependências de build.

## Publicar
Aponte a hospedagem (GitHub Pages, Netlify ou Vercel) para a raiz do repositório.
O arquivo servido é o `index.html`.
